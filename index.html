Keplerian Dynamics Examples

This is a repository containing three Pluto notebooks that are automatically converted to HTML by a github action, and published to github pages.

They explore a range of different calculations:

  • Part1.jl: Solution of Keplers Equation, Classical Orbital Elements to rv state function, Ephemeris function.

  • Part2.1.jl: Two-body dynamics orbit propagation.

  • Part2.2.jl: RV state vector to Classical Orbital Elements function, other frame conversion functions, maneuver calculations.
